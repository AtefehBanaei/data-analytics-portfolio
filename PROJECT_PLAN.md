\# Project Plan — E-commerce Sales \& Retention Analysis



\## 🎯 Project Goal

Analyze e-commerce sales data to understand revenue trends, customer behavior, and retention patterns.



This project is designed as a structured portfolio case study in two phases.



---



\# 🟢 Phase 1 — Python Analysis (Current Phase)



\## Objective

Perform core business analysis using Python (Pandas + Visualization).



\## Scope (What we WILL do)

\- Monthly Revenue Trend

\- Revenue Growth Rate

\- Average Order Value (AOV)

\- Repeat Purchase Rate

\- 3–5 Visualizations

\- Business Insights Summary



\## Out of Scope (What we will NOT do)

\- Machine Learning

\- Advanced Cohort Modeling

\- Predictive Models

\- A/B Testing



\## Definition of Done

Phase 1 is complete when:

\- Analysis notebook/script is clean

\- Visualizations are saved

\- README is updated with insights

\- Changes are committed and pushed to GitHub



---



\# 🔵 Phase 2 — SQL Version (Future Upgrade)



\## Objective

Reproduce core analysis using PostgreSQL.



\## Deliverables

\- SQL queries

\- Structured SQL file

\- Comparison: Python vs SQL



This phase will start only after Phase 1 is completed.
